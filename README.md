# Experiment Git Cloner

## Usage
```
usage: main.py [-h] [-n NUM_PROCESSES] input output

positional arguments:
  input                 input file
  output                output dir

optional arguments:
  -h, --help            show this help message and exit
  -n NUM_PROCESSES, --num_processes NUM_PROCESSES
                        number of processes
```
